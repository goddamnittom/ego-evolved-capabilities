# Update Evolution

This folder contains the **update_evolution.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `json`
- `os`

## Usage

You can import and use the components of this script in Python:
```python
import update_evolution.update_evolution
```

Alternatively, run it directly from the parent directory:
```bash
python -m update_evolution.update_evolution
```
