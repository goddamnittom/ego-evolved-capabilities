# Search Skills

This folder contains the **search_skills.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `os`
- `sys`

## Usage

You can import and use the components of this script in Python:
```python
import search_skills.search_skills
```

Alternatively, run it directly from the parent directory:
```bash
python -m search_skills.search_skills
```
