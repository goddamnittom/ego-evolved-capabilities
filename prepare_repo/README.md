# Prepare Repo

This folder contains the **prepare_repo.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `os`
- `shutil`

## Usage

You can import and use the components of this script in Python:
```python
import prepare_repo.prepare_repo
```

Alternatively, run it directly from the parent directory:
```bash
python -m prepare_repo.prepare_repo
```
