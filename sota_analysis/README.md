# Sota Analysis

This folder contains the **sota_analysis.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `json`
- `re`

## Usage

You can import and use the components of this script in Python:
```python
import sota_analysis.sota_analysis
```

Alternatively, run it directly from the parent directory:
```bash
python -m sota_analysis.sota_analysis
```
