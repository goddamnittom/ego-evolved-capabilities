# T3 Complexity Model

This folder contains the **t3_complexity_model.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `json`
- `math`

## Usage

You can import and use the components of this script in Python:
```python
import t3_complexity_model.t3_complexity_model
```

Alternatively, run it directly from the parent directory:
```bash
python -m t3_complexity_model.t3_complexity_model
```
