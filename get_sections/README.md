# Get Sections

This folder contains the **get_sections.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `pypdf`
- `re`

## Usage

You can import and use the components of this script in Python:
```python
import get_sections.get_sections
```

Alternatively, run it directly from the parent directory:
```bash
python -m get_sections.get_sections
```
