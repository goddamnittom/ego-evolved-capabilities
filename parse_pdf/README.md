# Parse Pdf

This folder contains the **parse_pdf.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `pypdf`

## Usage

You can import and use the components of this script in Python:
```python
import parse_pdf.parse_pdf
```

Alternatively, run it directly from the parent directory:
```bash
python -m parse_pdf.parse_pdf
```
