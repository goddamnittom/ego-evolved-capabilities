This folder contains the **ingest_all_sms.py** script, which is part of Ego's evolved capabilities workflow.

## Dependencies

## Usage

You can import and use the components of this script in Python:
```python

```

Alternatively, run it directly from the parent directory:
```bash
python -m ingest_all_sms.ingest_all_sms
```
