# Mct Telemetry T3

This folder contains the **mct_telemetry_t3.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `datetime`
- `json`

## Usage

You can import and use the components of this script in Python:
```python
import mct_telemetry_t3.mct_telemetry_t3
```

Alternatively, run it directly from the parent directory:
```bash
python -m mct_telemetry_t3.mct_telemetry_t3
```
