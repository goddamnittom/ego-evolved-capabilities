# Fetch Readmes

This folder contains the **fetch_readmes.py** script, which is part of Ego's evolved capabilities workflow.

## Overview

This script defines utility routines for the project.

## Dependencies

- `json`
- `requests`

## Usage

You can import and use the components of this script in Python:
```python
import fetch_readmes.fetch_readmes
```

Alternatively, run it directly from the parent directory:
```bash
python -m fetch_readmes.fetch_readmes
```
